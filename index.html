Crystal Collector


Directions

  1. Each of the 4 crystals have a hidden value between 1-12
  2. By pressing the start button you will be given a random number that you must try to add up to by pressing on the crystal buttons
  3. Each time you start a game all values will be reset and randomized
  4. The Reset Button will clear wins and losses and start the game over
